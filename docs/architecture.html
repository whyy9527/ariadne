Ariadne / System Architecture

A cross-service dependency graph, sized for a context window.

Static scanners extract the contract layer from every repo, TF-IDF scoring links related nodes across services, and a SQLite-only query path returns ranked clusters — small enough to fit in an AI conversation, deterministic enough to cite in a pull request.

Pluggable scanners

Seven built-ins cover GraphQL SDL, Spring controllers, Kafka topics and listeners, Spring clients, and TS gql`…` / axios / fetch.

Anything else ships as module.path:ClassName in ariadne.config.json — no Ariadne fork required.

Query is just SQLite

Scoring runs once at scan time and freezes into an edges table. Every query_chains call is a handful of indexed SELECTs — no model loading, no inference, no network.

Cluster confidence = edge score · 0.6 + type diversity · 0.2 + service diversity · 0.2.

Feedback learns the vocabulary

A follow-up expand_node within ten minutes auto-writes positive feedback — the expand is the signal.

Rerank adds 0.15 · Σ accepted on top of lexical confidence. Count-based, 90-day decay, local only. Disable with ARIADNE_FEEDBACK_BOOST=0.
